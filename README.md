# IDM Streams & BioSoul Essências - Painel Administrativo

Este repositório contém o código-fonte do painel de gerenciamento e servidor para os serviços integrados da BioSoul Essências, IDM Streams (acervo de músicas) e sistemas de afiliados (Mercado Livre/Shopee/Amazon), incluindo automação de anúncios e ganhos por visualizações/vendas.

## 🚀 Tecnologias Utilizadas

* **JavaScript (Node.js)** - Lógica do servidor e painel.
* **Firebase** - Banco de dados em tempo real e autenticação.
* **Mídias/APIs** - Integração com plataformas de afiliados e recargas.

## 📂 Estrutura do Projeto

* `servidor.js` - Arquivo principal que roda o servidor backend.
* `painel.js` - Lógica de controle do painel do usuário/administrador.
* `firebase.js` - Configuração e conexão com o banco de dados.
* `configuracoes.js` - Definições gerais do sistema.
* `.env` - Variáveis de ambiente e credenciais secretas (não enviado ao GitHub).
